对象.addEventListener("事件类型"，事件处理函数，boolean); 火狐和谷歌支持，IE8不支持。

对象.attachEvent("有on的事件类型"，事件处理函数);IE8支持，火狐谷歌不支持。

addEventListener和attachEvent的不同点和相同点

相同点都是可以为同一个元素绑定相同事件源的多个事件。

不同点：

  • 方法参数不一样.addEventListener("type",fn,false);attachEvent(on+"type",fn)
  • 在事件类型上，attachEvent需要添加on,例如click，需要加上on变为onclick。
  • addEventListener支持火狐、谷歌以及IE11，attachEvent支持IE8。
  • addEventListener函数里面的this指向的是事件源对象，attachEvent指向的window对象。
